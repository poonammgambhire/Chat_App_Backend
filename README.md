# 💬 Chat App Backend

Real-time Chat Application Backend built with Node.js & Socket.IO.

## 🚀 Features
- Real-time Messaging (Socket.IO)
- User Authentication (JWT + bcrypt)
- Group Chat & Private Chat
- Online/Offline Status
- Message History (MongoDB)
- Image Upload (Cloudinary)

## 🛠️ Tech Stack
**Backend:** Node.js, Express.js, MongoDB

**Real-time:** Socket.IO

**Other:** JWT, bcrypt, Cloudinary, Multer

## ⚙️ Installation

```bash
# Clone the repo
git clone https://github.com/poonammgambhire/Chat_App_Backend.git

# Install dependencies
npm install

# Run
npm start
```
